ABOUT ME

Hello! I'm a second-year Computer Science and Engineering student at UC Davis. My mission this year is to build projects that excite me and expand my interests, which currently include back-end development and geographic information systems. I'm currently working on making this website more responsive and building a Battleship experience in Javascript, based on a working C++ model I created last year. Let's connect!

Outside of school and work, you can find me reading tech blogs or watching YouTube tutorials to catch up on the latest coding trends in order to help me develop a growth mindset and become a more flexible developer. And when I'm not coding, you can find me painting my next "masterpiece" or logging my latest read on Goodreads.

FEATURED PROJECTS

Battleship

A multiplayer Battleship game written in C++ and run through the terminal. The game offers multiple AI/Human Player options and allows users to customize the dimensions of the playing board and ships (with error-checking).

The Anthology of...

A Python script that creates and fills Spotify playlists with a given artist's entire released works. I created this service as an extension of the "This is..." Spotify playlist that each artist receives. I'm currently working on optimizing the script for more popular artists and building a web application for the service to make it more user-friendly. Visit the Github page here!

Random Piano

A web app created with Javascript that plays randomly-animated piano keys when users type on the keyboard. This is the first web app I've created that relies on Javascript for functionality. I learned how to manipulate the DOM and how to use functional programming to implement proper event handling. In the future, I would like to experiment with different text passages to allow the users to "type music." Play here!
